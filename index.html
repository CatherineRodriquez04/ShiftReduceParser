Shift Reduce Parser:

Shift-reduce parsing is a form of bottom-up parsing in which a stack holds grammar symbols and an input buffer holds the rest of the tokens to be parse.


Created By Catherine Rodriquez: Spring 2024

Shift Reduce Parser

State Action Goto
id + * ( ) $ E T F

Production Rule

Input

1. E -> E + T
2. E -> T
3. T -> T * F
4. T -> F
5. F -> (E)
6. F -> id
